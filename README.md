# AppDynamicsDotNetSilentInstaller

This batch scripts peforms 3 main actions: 

1. check if AppDynamics Agent is installed 
2. Uninstall the agent if it installed 
3. Install a new agent 

All 3 actions are performed silently in the background. 

See here for details - https://docs.appdynamics.com/display/PRO45/Unattended+Installation+for+.NET
